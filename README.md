# Student Management System

A menu-driven Student Management System built using Python.

* Features
- Add Student
- View All Students
- Search Student
- Update Student
- Delete Student

* Concepts Used
- Python Functions
- Dictionaries
- Loops
- Conditional Statements
- CRUD Operations

* How to Run

```bash
python student_management.py
```

* Future Improvements

- File Handling
